# React Study

React를 학습하며 작성한 예제와 실습 코드를 정리한 저장소입니다.

## 📚 학습 내용

- React 기초
- JSX
- Component
- Props & State
- Event Handling
- Hooks
- React Router
- Axios
- 프로젝트 실습

## 🛠️ 개발 환경

- React
- Vite
- Node.js
- npm

## 📁 폴더 구조

```
src/
├── assets/
├── components/
├── pages/
├── App.jsx
└── main.jsx
```

## 🚀 실행 방법

```bash
npm install
npm run dev
```

## 🎯 목표

React의 기본 개념을 이해하고 컴포넌트 기반 개발 방식에 익숙해지는 것을 목표로 합니다.
